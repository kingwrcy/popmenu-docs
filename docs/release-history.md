# 发布历史

---

### 2025.08.12 v0.1

首次发布

- **多窗口支持**：任意多开，随心排列组合，信息一目了然。
- **内容完全自定义**：只需简单的 HTML+JS+CSS，想显示什么由你决定。
- **极致轻量**：压缩后仅 769KB，秒级启动，资源占用极低。
- **窗口自由拖拽**：支持拖动、置顶、关闭等常用操作。
- **一键切换内容**：右键菜单快速切换自定义页面。
- **开箱即用**：默认集成 `$v2ex` 币价。
